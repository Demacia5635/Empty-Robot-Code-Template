# Demacia_Lib 5635

ספריית התשתיות של קבוצה **Demacia 5635** לפיתוח רובוטים ב-FRC.

### ❓ מה זה?
זוהי מעטפת (Framework) מקיפה שנועדה לייעל ולפשט את כתיבת קוד הרובוט. הספרייה חוסכת זמן פיתוח יקר ומאפשרת לכתוב קוד נקי, קריא ואחיד.

### ✨ פיצ'רים עיקריים

🛠️ **ניהול מנועים מתקדם**
תמיכה מלאה ב-**Talon (FX/SRX)** וב-**Neo (Spark Max/Flex)**.
* הגדרה פשוטה ונוחה (Config Builder).
* המרות יחידות אוטומטיות (מטרים/רדיאנים).
* אינטגרציה מלאה עם **SysID**.
* ווידג'טים לכיול PID בזמן אמת דרך Elastic Dashboard.

ממשק אחיד לכל החיישנים הנפוצים ב-FRC:
* **Gyro:** Pigeon 2.0
* **Encoders:** CANcoder, Digital/Analog Encoders
* **Distance:** UltraSonic, Lidar, Optic
* **Digital:** Limit Switches

🏎️ **מערכות שאסי (Chassis)**
קוד מוכן לשימוש עבור מערכות הנעה (Swerve / Tank) עם ניהול אודומטריה ותנועה.

⚙️ **תשתית מנגנונים (Subsystems)**
מחלקות בסיס לניהול מנגנונים ומכונות מצבים (State Machines) בקלות.

📊 **טלמטריה ולוגים**
מערכת הוספת לוגים חכמה ופשוטה לניהול נתונים בזמן אמת ואחרי המשחק.

🎮 **בקרים ושליטה**
תשתיות מוכנות לשלטי **Xbox** ו-**PS5**.

💡 **תוספות שימושיות**
* **Lookup Table:** לחישובים מהירים.
* **LEDs:** ניהול תאורה חכם לרובוט.
